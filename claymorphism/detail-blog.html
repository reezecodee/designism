Cara Setup CI/CD Pipeline Jenkins untuk Project Laravel

Portfolio

14 Jan 2026

Deployment manual itu melelahkan. Bayangkan setiap kali ada update kecil di backend Laravel, kita harus login SSH, `git pull`, jalankan migrasi, dan restart service. Di artikel ini, saya akan berbagi pengalaman setup Jenkins untuk mengotomatisasi proses tersebut.

1. Persiapan Server

Pastikan server VPS (saya pakai Ubuntu 22.04) sudah terinstall Java, karena Jenkins berjalan di atas Java Runtime Environment (JRE). 

sudo apt update
sudo apt install fontconfig openjdk-17-jre
java -version

2. Install Jenkins

Selanjutnya, kita tambahkan repository Jenkins ke sistem dan lakukan instalasi. 

sudo apt-get install jenkins
sudo systemctl start jenkins
